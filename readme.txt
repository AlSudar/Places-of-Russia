Адаптивная страница свёрстанная при помощи SCSS и HTML5. Сборка происходит при помощи Parcel, сервер запускается ExpressJS.
Команда для запуска девелопа - npm run dev
Команда для билда - npm run build
Команда для продакшена - npm run serve

Responsive page made with SCSS and HTML5. The assembly is done using Parcel, the server is started by ExpressJS.
The command to run the developer - npm run dev
Build command - npm run build
Production command - npm run serve
