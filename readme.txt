Респонсивная страница свёрстанная при помощи CSS3 и HTML5. 
Файлы CSS были проименованы согласно БЭМ, разбиты по блоками и к .html подключаются импортом. 
В работе используется подключённый шрифт


Responsive page collapsed using CSS3 and HTML5.
CSS files were named according to BEM, broken into blocks and connected to .html by import.
The work uses a connected font
